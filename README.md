# Remax Wechat

自用

使用 Remax 开发微信小程序的云开发模板。
<br/>
在cloudfunctions中写云函数，
在src下写页面
<br/>
详情参考官方文档:https://remaxjs.org/guide/quick-start
## Getting Start

安装依赖

```bash
npm install
```

调试项目

```bash
# 执行调试命令
$ npm run dev
or
yarn dev
```

使用小程序开发者工具打开项目下的 `dist` 目录

## 构建

```bash
# 执行构建命令
$ npm run build
or
$ yarn build
```

使用小程序开发者工具打开项目目录，上传代码即可。
